## ✨ What is ModDev Validator?

**ModDev Validator** is a powerful, all-in-one code validation tool that checks your HTML, CSS, JavaScript, and JSON before they go live. No more surprise errors in production! 🎉

Think of it as your personal code quality guardian—always watching, always helping.

---

## 🎯 Features

### 🏷️ **HTML Validator**
- ✅ Checks for unclosed tags
- ✅ Detects invalid nesting
- ✅ Finds missing `<html>` and `<body>` tags
- ✅ Spots duplicate IDs
- 🌳 **DOM Tree Preview** - Visualize your HTML structure

### 🎨 **CSS Validator**
- ✅ Detects missing semicolons
- ✅ Validates property names
- ✅ Checks for invalid units (px, em, rem, %, etc.)
- ✅ Identifies syntax errors

### 📜 **JavaScript Checker**
- ✅ Catches syntax errors instantly
- ✅ Flags unused variables
- ✅ Warns about missing semicolons
- ✅ Sandboxed execution environment

### 📦 **JSON Validator/Formatter**
- ✅ Validates JSON structure
- ✅ Detects trailing commas
- ✅ Pretty-prints your JSON
- ✅ Shows error positions
